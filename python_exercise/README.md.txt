# Library Management System

## Description
This is a library management system implemented in Python using object-oriented programming. It supports book borrowing, returning, and dynamic book searches.

## Features
- View available books
- Add new books
- Borrow and return books
- Search for books
- Save and load library data

## How to Run
1. Clone the repository: `git clone <repo_link>`
2. Navigate to the `python_exercise` directory.
3. Run the program: `python library_system.py`
